Flow Description

🚀 Entry Point

  • App Launch leads to Main Screen with Camera View

📱 Main Screen Actions

  • Side Sheet: Access videos, coaching sessions, and app settings
  • Coach Tab: Chat with AI coach or use mirror mode
  • Native Media Picker: Direct access to media selection
  • Start Recording: Begin new video recording with motion capture
  • Insights Tab: View performance analysis and feedback
  • Upload Video (Left): Camera control to launch media picker
  • Camera Swap Control (Right): Switch between front/back camera

📷 Camera Controls

  • Upload Video → Native Media Picker → Select Video/File → Analysis Processing
  • Camera Swap Control: Direct return to Main Screen
  • Start Recording → Active State with Zoom In/Out controls
  • Pause/Resume functionality during recording
  • Stop Recording → Analysis Processing

🔔 Notifications System

  • Notifications Menu: Access notification management
  • View Notifications: Browse all notifications
  • Mark as Read: Mark individual notifications as read
  • Clear All: Remove all notifications
  • Notification Settings: Configure notification preferences

📊 Analysis & Feedback

  • AI Processing: Automatic video analysis
  • Feedback Review Screen: Video playback with overlay
  • AI Feedback Timeline: Time-synced coaching points
  • Performance Insights: Detailed analysis results
  • Play/Pause Controls: Interactive video navigation

🤖 Coach Interaction

  • Chat Mode: Text-based conversation with AI avatar
  • Mirror Mode: Future video call functionality
  • Type and send messages to AI coach
  • Receive animated avatar responses

📚 Video Management

  • View Video List: Browse all recorded sessions
  • Filter Screen: Sort and filter videos
  • Select Video: Choose specific recording
  • Video Detail View: Full playback experience
  • Playback with Feedback: AI insights overlay

⚙️ Settings Management

  • Account Settings, Personalisation, Give Feedback
  • Data Controls, Security, About
  • Log Out functionality
